# Mohammad Fakhruddin Babar — Personal Website

A responsive single-page academic portfolio built for GitHub Pages.

## Publish on GitHub Pages

1. Replace the files in the repository root with the contents of this folder.
2. Commit and push to the `main` branch.
3. In **Settings → Pages**, select **Deploy from a branch**, then choose `main` and `/ (root)`.

No build system is required. The site uses plain HTML, CSS, and JavaScript.
