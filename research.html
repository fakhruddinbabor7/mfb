Trusted Execution · Real-Time

Confidential DNN Execution Inside TrustZone

We developed a real-time scheduling framework for confidential DNN inference on ARM TrustZone-based secure enclaves. We built analytical models to evaluate whether real-time DNN workloads can meet timing constraints while executing inside Trusted Execution Environments (TEEs). Our approach can reduce TEE context-switch overhead while preserving real-time guarantees

Key points

  • Analytical schedulability models
  • Secure-memory-aware DNN deployment
  • Reduced trusted-execution context switches

Covert Channels and Information Leakage

We investigate how traditional scheduler can reveal sensitive execution patterns in fixed-priority real-time systems. We develops analytical model and statistical techniques that analyzes existence of covert channel and information leakgae.

Key points

Embedded AI · Performance Analysis

Language Models on Resource-constrained Devices

We analyze the feasibility of deploying language models in embedded devices such as Raspberry PI, Jetson, UP, and UDOO. We measure inference latency, memory usage, energy consumption, model size etc. and develop a lookup table that can assist a system designer to choose the best model for a given hardware requirements.

Key points

  • Measures latency, memory, energy, and accuracy
  • Compares models across embedded platforms
  • Identifies practical deployment tradeoffs

Vulnerability Prioritization

We develop a composite metrics that combine exploitability, severity, weakness information, threat intelligence, and semantic signals to help defenders prioritize CVEs and allocate remediation resources.

Key points

  • Supports earlier identification of high-risk CVEs
